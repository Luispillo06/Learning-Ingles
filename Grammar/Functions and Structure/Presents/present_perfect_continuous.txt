================================================================================
                      PRESENT PERFECT CONTINUOUS
                  (Presente Perfecto Continuo en Inglés)
================================================================================

El Present Perfect Continuous enfatiza la DURACIÓN de una actividad que 
empezó en el pasado y continúa hasta ahora (o acaba de terminar).

================================================================================
                         STRUCTURE (Estructura)
================================================================================

AFFIRMATIVE (Afirmativo):

┌─────────────────────────────────────────────────────────────────────────────┐
│  SUBJECT    +    HAVE/HAS BEEN    +    VERB-ING    +    COMPLEMENT          │
├─────────────────────────────────────────────────────────────────────────────┤
│  I/You/We/They    have been           working          all day.             │
│  He/She/It        has been            working          all day.             │
└─────────────────────────────────────────────────────────────────────────────┘

→ I have been studying for three hours.
→ She has been working here since 2020.
→ They have been waiting for an hour.
→ He has been learning English for two years.

CONTRACTIONS:
I have been = I've been           He has been = He's been
She has been = She's been         We have been = We've been

NEGATIVE (Negativo):

┌─────────────────────────────────────────────────────────────────────────────┐
│  SUBJECT    +    HAVEN'T/HASN'T BEEN    +    VERB-ING                       │
├─────────────────────────────────────────────────────────────────────────────┤
│  I/You/We/They    haven't been              working.                        │
│  He/She/It        hasn't been               working.                        │
└─────────────────────────────────────────────────────────────────────────────┘

→ I haven't been sleeping well lately.
→ She hasn't been feeling well.
→ They haven't been coming to class.
→ He hasn't been working this week.

INTERROGATIVE (Interrogativo):

┌─────────────────────────────────────────────────────────────────────────────┐
│  HAVE/HAS    +    SUBJECT    +    BEEN    +    VERB-ING    +    ?           │
├─────────────────────────────────────────────────────────────────────────────┤
│  Have             I/you/we/they   been         working?                     │
│  Has              he/she/it       been         working?                     │
└─────────────────────────────────────────────────────────────────────────────┘

→ Have you been waiting long?
→ Has she been crying?
→ What have you been doing?
→ How long have they been living here?

SHORT ANSWERS (Respuestas cortas):

→ Have you been working?     Yes, I have. / No, I haven't.
→ Has she been studying?     Yes, she has. / No, she hasn't.

================================================================================
                         USES (Usos)
================================================================================

1. DURATION OF AN ACTIVITY UP TO NOW (Duración hasta ahora):

┌───────────────────────────────────────────────────────────────────────────────┐
│ I have been working here for five years.                                      │
│ He estado trabajando aquí durante cinco años.                                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ She has been learning Spanish since January.                                  │
│ Ella ha estado aprendiendo español desde enero.                               │
├───────────────────────────────────────────────────────────────────────────────┤
│ They have been waiting for two hours.                                         │
│ Han estado esperando durante dos horas.                                       │
├───────────────────────────────────────────────────────────────────────────────┤
│ How long have you been living in this city?                                   │
│ ¿Cuánto tiempo llevas viviendo en esta ciudad?                                │
└───────────────────────────────────────────────────────────────────────────────┘

Keywords: for, since, how long, all day/morning/week

2. RECENT ACTIVITY WITH VISIBLE RESULTS (Actividad reciente con resultados visibles):

┌───────────────────────────────────────────────────────────────────────────────┐
│ You're out of breath. Have you been running?                                  │
│ Estás sin aliento. ¿Has estado corriendo?                                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ Your eyes are red. Have you been crying?                                      │
│ Tus ojos están rojos. ¿Has estado llorando?                                   │
├───────────────────────────────────────────────────────────────────────────────┤
│ The ground is wet. It has been raining.                                       │
│ El suelo está mojado. Ha estado lloviendo.                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ I'm tired because I have been working all day.                                │
│ Estoy cansado porque he estado trabajando todo el día.                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ She's dirty because she has been gardening.                                   │
│ Está sucia porque ha estado trabajando en el jardín.                          │
└───────────────────────────────────────────────────────────────────────────────┘

3. TEMPORARY SITUATIONS (Situaciones temporales):

┌───────────────────────────────────────────────────────────────────────────────┐
│ I've been staying with my parents while my house is being renovated.          │
│ Me he estado quedando con mis padres mientras mi casa está siendo renovada.   │
├───────────────────────────────────────────────────────────────────────────────┤
│ She's been working extra hours this month.                                    │
│ Ella ha estado trabajando horas extra este mes.                               │
├───────────────────────────────────────────────────────────────────────────────┤
│ They've been living in a hotel since the fire.                                │
│ Han estado viviendo en un hotel desde el incendio.                            │
└───────────────────────────────────────────────────────────────────────────────┘

4. REPEATED ACTIONS (Acciones repetidas):

┌───────────────────────────────────────────────────────────────────────────────┐
│ I've been calling you all morning.                                            │
│ Te he estado llamando toda la mañana.                                         │
├───────────────────────────────────────────────────────────────────────────────┤
│ She's been taking driving lessons.                                            │
│ Ella ha estado tomando clases de conducir.                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ He's been going to the gym regularly.                                         │
│ Ha estado yendo al gimnasio regularmente.                                     │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    KEY WORDS AND TIME EXPRESSIONS
================================================================================

FOR + DURATION:
    for two hours, for three days, for a week, for months, for years

SINCE + POINT IN TIME:
    since 2020, since Monday, since 9 o'clock, since I arrived

HOW LONG:
    How long have you been waiting?
    How long has she been working here?

ALL + TIME PERIOD:
    all day, all morning, all afternoon, all week, all year

LATELY / RECENTLY:
    I've been feeling tired lately.
    She hasn't been sleeping well recently.

================================================================================
          PRESENT PERFECT CONTINUOUS vs PRESENT PERFECT
================================================================================

┌──────────────────────────────────────┬────────────────────────────────────────┐
│ PRESENT PERFECT CONTINUOUS           │ PRESENT PERFECT                        │
├──────────────────────────────────────┼────────────────────────────────────────┤
│ Emphasizes DURATION                  │ Emphasizes COMPLETION/RESULT           │
│ I've been reading this book.         │ I've read this book.                   │
│ (still reading or just stopped)      │ (finished it)                          │
├──────────────────────────────────────┼────────────────────────────────────────┤
│ Focus on the ACTIVITY                │ Focus on the NUMBER/QUANTITY           │
│ I've been writing emails all day.    │ I've written 20 emails today.          │
│ (the activity itself)                │ (how many)                             │
├──────────────────────────────────────┼────────────────────────────────────────┤
│ TEMPORARY situations                 │ PERMANENT situations                   │
│ She's been living in London.         │ She has lived in London all her life.  │
│ (temporary, might change)            │ (permanent, stable)                    │
├──────────────────────────────────────┼────────────────────────────────────────┤
│ Action MAY be incomplete             │ Action is COMPLETE                     │
│ I've been painting the house.        │ I've painted the house.                │
│ (maybe not finished)                 │ (finished)                             │
└──────────────────────────────────────┴────────────────────────────────────────┘

MORE EXAMPLES TO COMPARE:

→ I've been cleaning the house. (todavía limpiando o acabo de parar)
→ I've cleaned the house. (terminé, está limpia)

→ She's been making a cake. (todavía haciéndola o recién terminó)
→ She's made a cake. (terminó, el pastel está listo)

→ How long have you been learning English? (duración del proceso)
→ How many languages have you learned? (cantidad/resultado)

→ I've been reading. (actividad en general)
→ I've read three books this month. (número específico)

================================================================================
                    VERBS NOT USED IN CONTINUOUS
================================================================================

Los STATIVE VERBS normalmente NO se usan en Present Perfect Continuous:

know, believe, understand, remember, want, need, like, love, hate,
own, belong, have (possession), be

En estos casos, usamos Present Perfect:

❌ I've been knowing her for years.    ✓ I've known her for years.
❌ She's been wanting to go.           ✓ She has wanted to go.
❌ They've been owning this house.     ✓ They've owned this house.
❌ I've been being tired.              ✓ I've been tired.

================================================================================
                    HOW LONG + PRESENT PERFECT CONTINUOUS
================================================================================

La pregunta más común con este tiempo es "HOW LONG...?":

┌───────────────────────────────────────────────────────────────────────────────┐
│ How long have you been learning English?                                      │
│ ¿Cuánto tiempo llevas aprendiendo inglés?                                     │
│ → I've been learning English for two years.                                   │
├───────────────────────────────────────────────────────────────────────────────┤
│ How long has she been working here?                                           │
│ ¿Cuánto tiempo lleva ella trabajando aquí?                                    │
│ → She's been working here since 2019.                                         │
├───────────────────────────────────────────────────────────────────────────────┤
│ How long have they been waiting?                                              │
│ ¿Cuánto tiempo llevan esperando?                                              │
│ → They've been waiting for over an hour.                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ How long has it been raining?                                                 │
│ ¿Cuánto tiempo lleva lloviendo?                                               │
│ → It's been raining since this morning.                                       │
└───────────────────────────────────────────────────────────────────────────────┘
