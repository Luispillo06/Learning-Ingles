================================================================================
                    VERBS WITH TWO OBJECTS
                    (Verbos con Dos Objetos)
================================================================================

Algunos verbos pueden tener DOS objetos:
- INDIRECT OBJECT (persona - a quién)
- DIRECT OBJECT (cosa - qué)

================================================================================
                    BASIC STRUCTURE
================================================================================

Two possible word orders:

┌───────────────────────────────────────────────────────────────────────────────┐
│ STRUCTURE 1: verb + indirect object + direct object                           │
│                                                                               │
│ She gave    ME         a present.                                             │
│             ──         ─────────                                              │
│             (to whom)  (what)                                                 │
│             indirect   direct                                                 │
│                                                                               │
│ Me dio un regalo.                                                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ STRUCTURE 2: verb + direct object + TO/FOR + indirect object                  │
│                                                                               │
│ She gave    a present    TO ME.                                               │
│             ─────────    ─────                                                │
│             (what)       (to whom)                                            │
│             direct       indirect                                             │
│                                                                               │
│ Me dio un regalo.                                                             │
└───────────────────────────────────────────────────────────────────────────────┘

Both structures mean the same thing!

================================================================================
                    TO vs FOR - THE DIFFERENCE
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│ TO = transfer/movement towards someone                                        │
│ (the person RECEIVES something)                                               │
│                                                                               │
│ Give the book TO me.       → I receive the book                               │
│ Send the letter TO her.    → She receives the letter                          │
│ Pass the salt TO him.      → He receives the salt                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ FOR = benefit/on behalf of someone                                            │
│ (something is done FOR someone's benefit)                                     │
│                                                                               │
│ Buy a coffee FOR me.       → for my benefit                                   │
│ Make a cake FOR her.       → for her benefit                                  │
│ Save a seat FOR him.       → for his benefit                                  │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    WHEN TO USE WHICH STRUCTURE
================================================================================

STRUCTURE 1 (verb + indirect + direct) is preferred when:
┌───────────────────────────────────────────────────────────────────────────────┐
│ ✓ The indirect object is SHORT (pronoun):                                     │
│                                                                               │
│ Give ME the book. ✓ (natural)                                                 │
│ Tell HER the truth. ✓ (natural)                                               │
│ Buy HIM a present. ✓ (natural)                                                │
└───────────────────────────────────────────────────────────────────────────────┘

STRUCTURE 2 (verb + direct + to/for + indirect) is preferred when:
┌───────────────────────────────────────────────────────────────────────────────┐
│ ✓ The indirect object is LONG:                                                │
│                                                                               │
│ Give the book to the woman in the red dress. ✓                                │
│ ❌ Give the woman in the red dress the book. (awkward)                        │
│                                                                               │
│ Send the email to all the members of the committee. ✓                         │
│ ❌ Send all the members of the committee the email. (awkward)                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ ✓ The direct object is a PRONOUN (it/them):                                   │
│                                                                               │
│ Give IT to me. ✓                                                              │
│ ❌ Give me it. (possible but less common)                                     │
│                                                                               │
│ Send THEM to her. ✓                                                           │
│ ❌ Send her them. (awkward)                                                   │
├───────────────────────────────────────────────────────────────────────────────┤
│ ✓ You want to EMPHASIZE who receives it:                                      │
│                                                                               │
│ I gave the money to HIM, not to her.                                          │
│ She made the cake for ME, not for you.                                        │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    VERBS THAT ONLY USE ONE STRUCTURE
================================================================================

Some verbs ONLY use structure 2 (with to/for):

ONLY + TO:
┌───────────────────────────────────────────────────────────────────────────────┐
│ EXPLAIN   │ ❌ Explain me the problem.                                        │
│           │ ✓  Explain the problem to me.                                     │
│           │    Explícame el problema.                                         │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ DESCRIBE  │ ❌ Describe me the situation.                                     │
│           │ ✓  Describe the situation to me.                                  │
│           │    Descríbeme la situación.                                       │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ SUGGEST   │ ❌ Suggest me a restaurant.                                       │
│           │ ✓  Suggest a restaurant to me.                                    │
│           │    Sugiéreme un restaurante.                                      │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ ANNOUNCE  │ ❌ Announce us the news.                                          │
│           │ ✓  Announce the news to us.                                       │
│           │    Anúncianos la noticia.                                         │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ MENTION   │ ❌ Mention me the problem.                                        │
│           │ ✓  Mention the problem to me.                                     │
│           │    Menciónale el problema.                                        │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ INTRODUCE │ ❌ Introduce me your friend.                                      │
│           │ ✓  Introduce your friend to me.                                   │
│           │    Preséntame a tu amigo.                                         │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ REPORT    │ ❌ Report me the results.                                         │
│           │ ✓  Report the results to me.                                      │
│           │    Repórtame los resultados.                                      │
├───────────┼───────────────────────────────────────────────────────────────────┤
│ SAY       │ ❌ Say me something.                                              │
│           │ ✓  Say something to me.                                           │
│           │    Dime algo.                                                     │
└───────────┴───────────────────────────────────────────────────────────────────┘

⚠️ These are common errors for Spanish speakers!

================================================================================
                    PASSIVE WITH TWO OBJECTS
================================================================================

Verbs with two objects can form TWO different passives:

Active: Someone gave me a book.

┌───────────────────────────────────────────────────────────────────────────────┐
│ PASSIVE 1 (person as subject) - MORE COMMON:                                  │
│                                                                               │
│ I was given a book.                                                           │
│ Me dieron un libro.                                                           │
├───────────────────────────────────────────────────────────────────────────────┤
│ PASSIVE 2 (thing as subject) - LESS COMMON:                                   │
│                                                                               │
│ A book was given to me.                                                       │
│ Un libro me fue dado.                                                         │
└───────────────────────────────────────────────────────────────────────────────┘

More examples:
┌───────────────────────────────────────────────────────────────────────────────┐
│ Active: They offered him a job.                                               │
│                                                                               │
│ Passive 1: He was offered a job. ✓ (more common)                              │
│            Le ofrecieron un trabajo.                                          │
│                                                                               │
│ Passive 2: A job was offered to him. (less common)                            │
│            Un trabajo le fue ofrecido.                                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ Active: The company paid her £1000.                                           │
│                                                                               │
│ Passive 1: She was paid £1000. ✓                                              │
│            Le pagaron £1000.                                                  │
│                                                                               │
│ Passive 2: £1000 was paid to her.                                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ Active: They sent me an invitation.                                           │
│                                                                               │
│ Passive 1: I was sent an invitation. ✓                                        │
│            Me enviaron una invitación.                                        │
│                                                                               │
│ Passive 2: An invitation was sent to me.                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ Active: They told us the news.                                                │
│                                                                               │
│ Passive 1: We were told the news. ✓                                           │
│            Nos contaron la noticia.                                           │
│                                                                               │
│ Passive 2: The news was told to us.                                           │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    QUICK SUMMARY
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│ VERBS + TO                        │ VERBS + FOR                              │
│ (transfer/receive)                │ (benefit/on behalf of)                   │
├───────────────────────────────────┼──────────────────────────────────────────┤
│ give, send, lend, pass, show,     │ buy, get, make, cook, find, save,        │
│ tell, teach, write, read, offer,  │ book, order, leave, fetch, pour,         │
│ promise, sell, pay, owe, bring,   │ build, design                            │
│ take, hand, throw                 │                                          │
├───────────────────────────────────┴──────────────────────────────────────────┤
│ ONLY STRUCTURE 2 (verb + direct + to):                                       │
│ explain, describe, suggest, announce, mention, introduce, report, say        │
│                                                                              │
│ ❌ Explain me... → ✓ Explain to me...                                        │
└──────────────────────────────────────────────────────────────────────────────┘
