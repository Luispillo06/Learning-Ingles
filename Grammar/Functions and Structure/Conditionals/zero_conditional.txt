================================================================================
                          ZERO CONDITIONAL
                   (Condicional Cero en Inglés)
================================================================================

El Zero Conditional se usa para hablar de VERDADES GENERALES, hechos
científicos y situaciones que SIEMPRE son verdad.

================================================================================
                         STRUCTURE (Estructura)
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│  IF    +    PRESENT SIMPLE    ,    PRESENT SIMPLE                           │
├─────────────────────────────────────────────────────────────────────────────┤
│  If         you heat water,         it boils.                               │
│  If         it rains,               the grass gets wet.                     │
└─────────────────────────────────────────────────────────────────────────────┘

También puede invertirse el orden:

┌─────────────────────────────────────────────────────────────────────────────┐
│  PRESENT SIMPLE    +    IF    +    PRESENT SIMPLE                           │
├─────────────────────────────────────────────────────────────────────────────┤
│  Water boils            if          you heat it.                            │
│  The grass gets wet     if          it rains.                               │
└─────────────────────────────────────────────────────────────────────────────┘

⚠️ NOTA: La coma se usa cuando IF va al principio.

================================================================================
                         USES (Usos)
================================================================================

1. SCIENTIFIC FACTS (Hechos científicos):

┌───────────────────────────────────────────────────────────────────────────────┐
│ If you heat water to 100°C, it boils.                                         │
│ Si calientas agua a 100°C, hierve.                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ If you freeze water, it becomes ice.                                          │
│ Si congelas agua, se convierte en hielo.                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ If you mix red and blue, you get purple.                                      │
│ Si mezclas rojo y azul, obtienes morado.                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ If there is no oxygen, fire goes out.                                         │
│ Si no hay oxígeno, el fuego se apaga.                                         │
└───────────────────────────────────────────────────────────────────────────────┘

2. GENERAL TRUTHS (Verdades generales):

┌───────────────────────────────────────────────────────────────────────────────┐
│ If you don't eat, you get hungry.                                             │
│ Si no comes, te da hambre.                                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ If babies are tired, they cry.                                                │
│ Si los bebés están cansados, lloran.                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ If you touch fire, you get burned.                                            │
│ Si tocas el fuego, te quemas.                                                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ If it rains, the streets get wet.                                             │
│ Si llueve, las calles se mojan.                                               │
└───────────────────────────────────────────────────────────────────────────────┘

3. INSTRUCTIONS AND RULES (Instrucciones y reglas):

┌───────────────────────────────────────────────────────────────────────────────┐
│ If the light is red, stop.                                                    │
│ Si la luz está roja, para.                                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ If you press this button, the machine starts.                                 │
│ Si pulsas este botón, la máquina arranca.                                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ If you see an error, restart the computer.                                    │
│ Si ves un error, reinicia el ordenador.                                       │
├───────────────────────────────────────────────────────────────────────────────┤
│ If the alarm goes off, leave the building.                                    │
│ Si suena la alarma, sal del edificio.                                         │
└───────────────────────────────────────────────────────────────────────────────┘

4. HABITS AND ROUTINES (Hábitos y rutinas):

┌───────────────────────────────────────────────────────────────────────────────┐
│ If I have time, I go to the gym.                                              │
│ Si tengo tiempo, voy al gimnasio.                                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ If she's stressed, she eats chocolate.                                        │
│ Si está estresada, come chocolate.                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ If my dad cooks, we order pizza.                                              │
│ Si mi padre cocina, pedimos pizza. (porque cocina mal)                        │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    IF vs WHEN
================================================================================

En el Zero Conditional, IF y WHEN son intercambiables:

┌───────────────────────────────────────────────────────────────────────────────┐
│ If/When you heat ice, it melts.                                               │
│ Si/Cuando calientas hielo, se derrite.                                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ If/When I'm tired, I go to bed early.                                         │
│ Si/Cuando estoy cansado, me acuesto temprano.                                 │
└───────────────────────────────────────────────────────────────────────────────┘

├──────────────────────────────────────────────────────────────────────────────┤
│ STRUCTURE:  If + Present Simple, Present Simple                              │
├──────────────────────────────────────────────────────────────────────────────┤
│ USE:        Facts that are ALWAYS true                                       │
│             • Scientific facts                                               │
│             • General truths                                                 │
│             • Instructions                                                   │
│             • Habits                                                         │
├──────────────────────────────────────────────────────────────────────────────┤
│ EXAMPLE:    If you heat water, it boils.                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│ NOTE:       IF and WHEN are interchangeable                                  │
└──────────────────────────────────────────────────────────────────────────────┘
