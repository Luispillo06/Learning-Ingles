================================================================================
                    HAVE SOMETHING DONE
                      (Causative)
================================================================================

Usamos HAVE SOMETHING DONE cuando otra persona hace algo por nosotros,
normalmente un servicio profesional que pagamos.

Structure: HAVE + OBJECT + PAST PARTICIPLE

================================================================================
                    BASIC STRUCTURE
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│                                                                               │
│     SUBJECT + HAVE + OBJECT + PAST PARTICIPLE                                 │
│                                                                               │
│     I        have   my car    repaired.                                       │
│              ────   ──────    ────────                                        │
│              verb   thing     done by someone else                            │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

COMPARE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ I repaired my car.                                                            │
│ Reparé mi coche. (yo mismo lo hice)                                           │
│                                                                               │
│ I had my car repaired.                                                        │
│ Hice que me repararan el coche. / Me repararon el coche.                      │
│ (alguien lo hizo por mí - un mecánico)                                        │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    GET SOMETHING DONE
================================================================================

GET SOMETHING DONE = HAVE SOMETHING DONE (more informal)

Structure: GET + OBJECT + PAST PARTICIPLE

┌───────────────────────────────────────────────────────────────────────────────┐
│ I need to get my car fixed.        (= have my car fixed)                      │
│ Necesito que me arreglen el coche.                                            │
│                                                                               │
│ She got her hair done yesterday.   (= had her hair done)                      │
│ Se arregló el pelo ayer.                                                      │
│                                                                               │
│ I'm getting my windows cleaned.    (= I'm having my windows cleaned)          │
│ Me están limpiando las ventanas.                                              │
│                                                                               │
│ You should get your eyes tested.   (= have your eyes tested)                  │
│ Deberías hacerte revisar la vista.                                            │
└───────────────────────────────────────────────────────────────────────────────┘

GET is more common in:
- Informal spoken English
- American English
- When talking about achieving something with effort

================================================================================
                    NEGATIVE EXPERIENCES
================================================================================

We also use HAVE SOMETHING DONE for bad things that happen to us
(things we don't arrange - accidents, crimes, etc.):

┌───────────────────────────────────────────────────────────────────────────────┐
│ I had my wallet stolen.                                                       │
│ Me robaron la cartera.                                                        │
│                                                                               │
│ She had her car broken into.                                                  │
│ Le entraron a robar en el coche.                                              │
│                                                                               │
│ He had his nose broken in a fight.                                            │
│ Le rompieron la nariz en una pelea.                                           │
│                                                                               │
│ They had their house burgled.                                                 │
│ Les robaron en casa.                                                          │
│                                                                               │
│ I had my phone hacked.                                                        │
│ Me hackearon el teléfono.                                                     │
│                                                                               │
│ She had her identity stolen.                                                  │
│ Le robaron la identidad.                                                      │
│                                                                               │
│ He had his leg broken in the accident.                                        │
│ Se rompió la pierna en el accidente.                                          │
└───────────────────────────────────────────────────────────────────────────────┘

⚠️ Context tells us if it's a service or a negative experience:
┌───────────────────────────────────────────────────────────────────────────────┐
│ I had my window broken.                                                       │
│                                                                               │
│ → Could mean: Someone broke it (crime/accident)                               │
│ → Or:         I paid someone to break it (unlikely but possible)              │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    HAVE/GET + OBJECT + INFINITIVE
================================================================================

Different meaning: to persuade/cause someone to do something

HAVE + OBJECT + BARE INFINITIVE (without to):
┌───────────────────────────────────────────────────────────────────────────────┐
│ I'll have my assistant call you.                                              │
│ Haré que mi asistente te llame.                                               │
│                                                                               │
│ She had the waiter bring more water.                                          │
│ Hizo que el camarero trajera más agua.                                        │
│                                                                               │
│ The teacher had the students write an essay.                                  │
│ El profesor hizo que los estudiantes escribieran un ensayo.                   │
└───────────────────────────────────────────────────────────────────────────────┘

GET + OBJECT + TO + INFINITIVE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ I got my brother to help me move.                                             │
│ Conseguí que mi hermano me ayudara con la mudanza.                            │
│                                                                               │
│ She got her boss to give her a raise.                                         │
│ Consiguió que su jefe le diera un aumento.                                    │
│                                                                               │
│ How did you get them to agree?                                                │
│ ¿Cómo conseguiste que aceptaran?                                              │
└───────────────────────────────────────────────────────────────────────────────┘

COMPARE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ I had my car repaired.        (past participle - service)                     │
│ → A mechanic repaired it for me (I paid)                                      │
│                                                                               │
│ I had my brother repair my car. (bare infinitive - causative)                 │
│ → I asked/told my brother and he did it                                       │
│                                                                               │
│ I got my brother to repair my car. (to + infinitive - persuasion)             │
│ → I persuaded my brother to do it                                             │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    ADDING THE AGENT (BY)
================================================================================

We can add who does the action (though it's often not necessary):

┌───────────────────────────────────────────────────────────────────────────────┐
│ I had my hair cut by a famous hairdresser.                                    │
│ Me cortó el pelo un peluquero famoso.                                         │
│                                                                               │
│ She had her portrait painted by a local artist.                               │
│ Le pintó el retrato un artista local.                                         │
│                                                                               │
│ They had their house designed by an architect.                                │
│ Les diseñó la casa un arquitecto.                                             │
└───────────────────────────────────────────────────────────────────────────────┘

Usually we DON'T include the agent when:
- It's obvious (hairdresser, mechanic, dentist)
- We don't know who did it
- It's not important

================================================================================
                    QUICK SUMMARY
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│ STRUCTURE          │ MEANING                                                  │
├────────────────────┼──────────────────────────────────────────────────────────┤
│ have + obj + pp    │ Someone does a service for you                           │
│                    │ I had my car repaired. (mechanic did it)                 │
├────────────────────┼──────────────────────────────────────────────────────────┤
│ get + obj + pp     │ Same as above (more informal)                            │
│                    │ I got my car repaired.                                   │
├────────────────────┼──────────────────────────────────────────────────────────┤
│ have + obj + pp    │ Something bad happens to you                             │
│ (negative)         │ I had my wallet stolen.                                  │
├────────────────────┼──────────────────────────────────────────────────────────┤
│ have + obj + inf   │ Cause/tell someone to do something                       │
│                    │ I had my assistant call you.                             │
├────────────────────┼──────────────────────────────────────────────────────────┤
│ get + obj + to inf │ Persuade someone to do something                         │
│                    │ I got my brother to help me.                             │
└───────────────────────────────────────────────────────────────────────────────┘

pp = past participle
inf = infinitive (without to for HAVE, with to for GET)
