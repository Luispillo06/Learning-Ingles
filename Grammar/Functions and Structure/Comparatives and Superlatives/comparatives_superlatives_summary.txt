================================================================================
                    FORMATION RULES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ ADJECTIVE TYPE            │ COMPARATIVE         │ SUPERLATIVE               │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1 syllable                │ + ER                │ THE + EST                 │
│ tall                      │ taller              │ the tallest               │
│ fast                      │ faster              │ the fastest               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Ending in -E              │ + R                 │ THE + ST                  │
│ nice                      │ nicer               │ the nicest                │
│ large                     │ larger              │ the largest               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Consonant + Y             │ Y → IER             │ THE + Y → IEST            │
│ happy                     │ happier             │ the happiest              │
│ easy                      │ easier              │ the easiest               │
├─────────────────────────────────────────────────────────────────────────────┤
│ CVC pattern (short)       │ double + ER         │ THE + double + EST        │
│ big                       │ bigger              │ the biggest               │
│ hot                       │ hotter              │ the hottest               │
├─────────────────────────────────────────────────────────────────────────────┤
│ 2+ syllables              │ MORE +              │ THE MOST +                │
│ beautiful                 │ more beautiful      │ the most beautiful        │
│ expensive                 │ more expensive      │ the most expensive        │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    IRREGULAR FORMS
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ ADJECTIVE       │ COMPARATIVE        │ SUPERLATIVE                         │
├─────────────────────────────────────────────────────────────────────────────┤
│ good            │ better             │ the best                            │
│ bad             │ worse              │ the worst                           │
│ far             │ farther/further    │ the farthest/furthest               │
│ little          │ less               │ the least                           │
│ much/many       │ more               │ the most                            │
│ old             │ older/elder        │ the oldest/eldest                   │
└─────────────────────────────────────────────────────────────────────────────┘

FARTHER vs FURTHER:
→ farther = physical distance (distancia física)
→ further = additional, more (adicional, más)
  "We need further information." (más información)

OLDER vs ELDER:
→ older = age in general (edad general)
→ elder = family members only (solo familia)
  "My elder brother" / "the eldest son"

================================================================================
                    STRUCTURES
================================================================================

COMPARATIVE (comparing 2):

┌───────────────────────────────────────────────────────────────────────────────┐
│ ADJECTIVE + ER + THAN          │ She is taller than me.                      │
│ MORE + ADJECTIVE + THAN        │ She is more beautiful than her sister.      │
└───────────────────────────────────────────────────────────────────────────────┘

SUPERLATIVE (comparing 3+):

┌───────────────────────────────────────────────────────────────────────────────┐
│ THE + ADJECTIVE + EST + IN/OF  │ She is the tallest in the class.            │
│ THE MOST + ADJECTIVE + IN/OF   │ This is the most expensive of all.          │
└───────────────────────────────────────────────────────────────────────────────┘

EQUALITY (igual):

┌───────────────────────────────────────────────────────────────────────────────┐
│ AS + ADJECTIVE + AS            │ She is as tall as her mother.               │
│ NOT AS + ADJECTIVE + AS        │ He isn't as fast as his brother.            │
└───────────────────────────────────────────────────────────────────────────────┘

INFERIORITY (menos):

┌───────────────────────────────────────────────────────────────────────────────┐
│ LESS + ADJECTIVE + THAN        │ This is less expensive than that one.       │
│ THE LEAST + ADJECTIVE          │ This is the least expensive option.         │
└───────────────────────────────────────────────────────────────────────────────┘

                    SPECIAL STRUCTURES

THE... THE... (Cuanto más... más...):
┌───────────────────────────────────────────────────────────────────────────────┐
│ The more you practice, the better you get.                                    │
│ Cuanto más practicas, mejor te vuelves.                                       │
├───────────────────────────────────────────────────────────────────────────────┤
│ The harder you work, the more money you earn.                                 │
│ Cuanto más trabajas, más dinero ganas.                                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ The sooner, the better.                                                       │
│ Cuanto antes, mejor.                                                          │
└───────────────────────────────────────────────────────────────────────────────┘

...ER AND ...ER / MORE AND MORE (Cada vez más):
┌───────────────────────────────────────────────────────────────────────────────┐
│ It's getting hotter and hotter.                                               │
│ Cada vez hace más calor.                                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ She's becoming more and more famous.                                          │
│ Se está volviendo cada vez más famosa.                                        │
└───────────────────────────────────────────────────────────────────────────────┘

ONE OF THE + SUPERLATIVE + PLURAL:
┌───────────────────────────────────────────────────────────────────────────────┐
│ He's one of the best players in the world.                                    │
│ Es uno de los mejores jugadores del mundo.                                    │
└───────────────────────────────────────────────────────────────────────────────┘

SUPERLATIVE + EVER + PRESENT PERFECT:
┌───────────────────────────────────────────────────────────────────────────────┐
│ This is the best movie I've ever seen.                                        │
│ Esta es la mejor película que he visto en mi vida.                            │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    MODIFIERS (Modificadores)
================================================================================

INTENSIFIERS (para comparativos):
┌───────────────────────────────────────────────────────────────────────────────┐
│ much / a lot / far + comparative = mucho más                                  │
│ She's much taller than me. | This is far more expensive.                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ a bit / a little / slightly + comparative = un poco más                       │
│ He's a bit older than me. | It's slightly cheaper.                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ even + comparative = incluso más                                              │
│ She's even more beautiful in person.                                          │
└───────────────────────────────────────────────────────────────────────────────┘

INTENSIFIERS (para superlativos):
┌───────────────────────────────────────────────────────────────────────────────┐
│ by far + superlative = con diferencia                                         │
│ He's by far the best player.                                                  │
├───────────────────────────────────────────────────────────────────────────────┤
│ easily + superlative = fácilmente                                             │
│ This is easily the most difficult exam.                                       │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    COMMON MISTAKES
================================================================================

❌ more taller / most tallest       ✓ taller / the tallest
❌ more better / most best          ✓ better / the best
❌ gooder / goodest                 ✓ better / the best
❌ badder / baddest                 ✓ worse / the worst
❌ more easy                        ✓ easier
❌ beautifuler                      ✓ more beautiful
❌ one of the best player           ✓ one of the best players
❌ taller that me                   ✓ taller than me
❌ the more expensive               ✓ the most expensive
❌ more cheap                       ✓ cheaper
