================================================================================
                    INDEFINITE ARTICLES: A / AN
                   (Artículos Indefinidos: A / AN)
================================================================================

A y AN significan "un/una". Se usan con sustantivos SINGULARES CONTABLES
cuando hablamos de algo NO ESPECÍFICO o mencionado por PRIMERA VEZ.

================================================================================
                    A vs AN - THE RULE
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ A  → before CONSONANT SOUNDS                                                │
│ AN → before VOWEL SOUNDS                                                    │
│                                                                             │
│ ⚠️ It's about the SOUND, not the letter!                                    │
└─────────────────────────────────────────────────────────────────────────────┘

A + CONSONANT SOUNDS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ a book        a car         a dog         a friend                            │
│ a house       a job         a student     a table                             │
│ a woman       a year        a zoo         a beautiful day                     │
└───────────────────────────────────────────────────────────────────────────────┘

AN + VOWEL SOUNDS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ an apple      an egg        an idea       an orange                           │
│ an umbrella   an elephant   an island     an office                           │
│ an interesting book         an old man    an easy question                    │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    EXCEPTIONS (SOUND, NOT SPELLING!)
================================================================================

A + words starting with vowel LETTER but consonant SOUND:

┌───────────────────────────────────────────────────────────────────────────────┐
│ a university   /juːnɪˈvɜːsɪti/    (starts with "y" sound)                     │
│ a uniform      /ˈjuːnɪfɔːm/       (starts with "y" sound)                     │
│ a European     /jʊərəˈpiːən/      (starts with "y" sound)                     │
│ a unit         /ˈjuːnɪt/          (starts with "y" sound)                     │
│ a one-way street  /wʌn/          (starts with "w" sound)                     │
│ a once-in-a-lifetime opportunity                                              │
└───────────────────────────────────────────────────────────────────────────────┘

AN + words starting with consonant LETTER but vowel SOUND:

┌───────────────────────────────────────────────────────────────────────────────┐
│ an hour        /aʊə/              (silent H)                                  │
│ an honest man  /ˈɒnɪst/           (silent H)                                  │
│ an heir        /eə/               (silent H)                                  │
│ an honour      /ˈɒnə/             (silent H)                                  │
│ an MBA         /em biː eɪ/        (starts with "e" sound)                     │
│ an FBI agent   /ef biː aɪ/        (starts with "e" sound)                     │
│ an X-ray       /eks/              (starts with "e" sound)                     │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    WHEN TO USE A/AN
================================================================================

1. FIRST MENTION (introducing something new):
┌───────────────────────────────────────────────────────────────────────────────┐
│ I saw a dog in the park. The dog was very friendly.                           │
│ Vi un perro en el parque. El perro era muy amigable.                          │
│ (first mention = a dog, second mention = the dog)                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ She bought a car. The car is red.                                             │
│ Ella compró un coche. El coche es rojo.                                       │
└───────────────────────────────────────────────────────────────────────────────┘

2. ONE OF MANY (not specific):
┌───────────────────────────────────────────────────────────────────────────────┐
│ I need a pen. (any pen, doesn't matter which one)                             │
│ Necesito un bolígrafo.                                                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ Can I have a glass of water?                                                  │
│ ¿Puedo tomar un vaso de agua?                                                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ Is there a bank near here?                                                    │
│ ¿Hay un banco cerca de aquí?                                                  │
└───────────────────────────────────────────────────────────────────────────────┘

3. JOBS AND PROFESSIONS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ She's a doctor. / He's an engineer.                                           │
│ Ella es médica. / Él es ingeniero.                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ I want to be a teacher.                                                       │
│ Quiero ser profesor.                                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ My father works as an accountant.                                             │
│ Mi padre trabaja como contable.                                               │
└───────────────────────────────────────────────────────────────────────────────┘

4. WITH "WHAT" IN EXCLAMATIONS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ What a beautiful day!                                                         │
│ ¡Qué día tan bonito!                                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ What a surprise!                                                              │
│ ¡Qué sorpresa!                                                                │
├───────────────────────────────────────────────────────────────────────────────┤
│ What an interesting idea!                                                     │
│ ¡Qué idea tan interesante!                                                    │
└───────────────────────────────────────────────────────────────────────────────┘

5. MEANING "PER" (rates/prices):
┌───────────────────────────────────────────────────────────────────────────────┐
│ 60 miles an hour = 60 miles per hour                                          │
│ 60 millas por hora                                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ twice a week = twice per week                                                 │
│ dos veces por semana                                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ $10 a kilo = $10 per kilo                                                     │
│ 10 dólares el kilo                                                            │
└───────────────────────────────────────────────────────────────────────────────┘

6. WITH CERTAIN NUMBERS AND QUANTITIES:
┌───────────────────────────────────────────────────────────────────────────────┐
│ a hundred / a thousand / a million                                            │
│ cien / mil / un millón                                                        │
├───────────────────────────────────────────────────────────────────────────────┤
│ a dozen eggs                                                                  │
│ una docena de huevos                                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ a couple of days                                                              │
│ un par de días                                                                │
├───────────────────────────────────────────────────────────────────────────────┤
│ half a kilo / a quarter of an hour                                            │
│ medio kilo / un cuarto de hora                                                │
└───────────────────────────────────────────────────────────────────────────────┘

7. WITH "SUCH", "QUITE", "RATHER":
┌───────────────────────────────────────────────────────────────────────────────┐
│ such a nice person                                                            │
│ una persona tan agradable                                                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ quite a long time                                                             │
│ bastante tiempo                                                               │
├───────────────────────────────────────────────────────────────────────────────┤
│ rather a difficult question                                                   │
│ una pregunta bastante difícil                                                 │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    WHEN NOT TO USE A/AN
================================================================================

❌ WITH PLURAL NOUNS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ ❌ I like a dogs.        →  ✓ I like dogs.                                    │
│ ❌ She has a books.      →  ✓ She has books.                                  │
└───────────────────────────────────────────────────────────────────────────────┘

❌ WITH UNCOUNTABLE NOUNS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ ❌ I need an advice.     →  ✓ I need advice. / I need some advice.            │
│ ❌ I want a water.       →  ✓ I want water. / I want a glass of water.        │
│ ❌ She has a money.      →  ✓ She has money.                                  │
│ ❌ I like a music.       →  ✓ I like music.                                   │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                           QUICK SUMMARY
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                           A / AN                                             │
├──────────────────────────────────────────────────────────────────────────────┤
│ A  → consonant SOUNDS: a book, a university, a one-way street                │
│ AN → vowel SOUNDS: an apple, an hour, an MBA                                 │
├──────────────────────────────────────────────────────────────────────────────┤
│ USE A/AN:                                                                    │
│ • First mention: I saw a dog.                                                │
│ • One of many: I need a pen.                                                 │
│ • Jobs: She's a doctor.                                                      │
│ • Exclamations: What a day!                                                  │
│ • Per: 60 miles an hour                                                      │
│ • Numbers: a hundred, a dozen                                                │
├──────────────────────────────────────────────────────────────────────────────┤
│ DON'T USE A/AN:                                                              │
│ • Plural nouns: I like dogs. (NOT a dogs)                                    │
│ • Uncountable: I need advice. (NOT an advice)                                │
└──────────────────────────────────────────────────────────────────────────────┘
