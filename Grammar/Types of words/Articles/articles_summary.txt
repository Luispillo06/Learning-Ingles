================================================================================
                    THE THREE ARTICLES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│  A / AN          │  THE              │  Ø (ZERO ARTICLE)                    │
│  Indefinite      │  Definite         │  No article                          │
│  Indefinido      │  Definido         │  Sin artículo                        │
├─────────────────────────────────────────────────────────────────────────────┤
│  un, una         │  el, la, los, las │  (nothing in English,                │
│                  │                   │   but article in Spanish)            │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    A vs AN
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ A  → before CONSONANT SOUNDS:  a book, a university, a one-way street      │
│ AN → before VOWEL SOUNDS:      an apple, an hour, an MBA                   │
│                                                                             │
│ ⚠️ It's about the SOUND, not the spelling!                                  │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    WHEN TO USE EACH ARTICLE
================================================================================

┌─────────────────┬────────────────────────────────────────────────────────────┐
│ ARTICLE         │ USE                                                        │
├─────────────────┼────────────────────────────────────────────────────────────┤
│ A / AN          │ First mention: I saw A dog.                                │
│                 │ One of many (not specific): I need A pen.                  │
│                 │ Jobs: She's A doctor.                                      │
│                 │ Exclamations: What A day!                                  │
│                 │ Per: 60 miles AN hour                                      │
│                 │ Numbers: A hundred, A dozen                                │
├─────────────────┼────────────────────────────────────────────────────────────┤
│ THE             │ Already mentioned: I saw a dog. THE dog was brown.         │
│                 │ Specific/known: THE book I gave you                        │
│                 │ Unique things: THE sun, THE moon                           │
│                 │ Superlatives: THE best, THE most beautiful                 │
│                 │ Ordinals: THE first, THE last                              │
│                 │ Rivers/seas: THE Thames, THE Atlantic                      │
│                 │ Instruments: play THE piano                                │
│                 │ Nationalities: THE Spanish, THE French                     │
├─────────────────┼────────────────────────────────────────────────────────────┤
│ Ø (NO ARTICLE)  │ Plural (general): (Ø) Dogs are loyal.                      │
│                 │ Uncountable (general): (Ø) Water is essential.             │
│                 │ Meals: have (Ø) breakfast                                  │
│                 │ Languages: speak (Ø) English                               │
│                 │ Sports: play (Ø) tennis                                    │
│                 │ Countries/cities: live in (Ø) Spain                        │
│                 │ Transport: by (Ø) bus                                      │
│                 │ Places (function): go to (Ø) school                        │
└─────────────────┴────────────────────────────────────────────────────────────┘

================================================================================
                    INSTITUTIONS AND PLACES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ FUNCTION (no article):               │ BUILDING (with THE):                 │
├──────────────────────────────────────┼──────────────────────────────────────┤
│ go to school (to study)              │ The school is old. (building)        │
│ go to church (to pray)               │ The church was built in 1800.        │
│ go to hospital (for treatment)       │ I visited the hospital.              │
│ be in prison (as prisoner)           │ The prison is outside town.          │
│ go to bed (to sleep)                 │ The bed is comfortable.              │
│ go to work (to work)                 │ -                                    │
│ be at home                           │ The home was decorated nicely.       │
└──────────────────────────────────────┴──────────────────────────────────────┘

================================================================================
                    GENERAL vs SPECIFIC
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│ GENERAL (no article):                                                         │
│ I like (Ø) dogs. (all dogs, in general)                                       │
│ (Ø) Coffee is delicious. (coffee in general)                                  │
│ (Ø) Life is short. (life in general)                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ SPECIFIC (with THE):                                                          │
│ I like THE dogs next door. (specific dogs)                                    │
│ THE coffee you made is delicious. (specific coffee)                           │
│ THE life of a doctor is difficult. (specific aspect)                          │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    COMMON COMBINATIONS
================================================================================

A/AN + JOB:
┌───────────────────────────────────────────────────────────────────────────────┐
│ She's a doctor. / He's an engineer. / I'm a student.                          │
└───────────────────────────────────────────────────────────────────────────────┘

THE + SUPERLATIVE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the best, the worst, the most expensive, the least important                  │
└───────────────────────────────────────────────────────────────────────────────┘

THE + ORDINAL:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the first, the second, the last, the next, the only                           │
└───────────────────────────────────────────────────────────────────────────────┘

THE + INSTRUMENT:
┌───────────────────────────────────────────────────────────────────────────────┐
│ play the piano, play the guitar, play the violin                              │
└───────────────────────────────────────────────────────────────────────────────┘

Ø + SPORT:
┌───────────────────────────────────────────────────────────────────────────────┐
│ play football, play tennis, play basketball                                   │
└───────────────────────────────────────────────────────────────────────────────┘

BY + Ø + TRANSPORT:
┌───────────────────────────────────────────────────────────────────────────────┐
│ by bus, by car, by train, by plane, on foot                                   │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    COMMON MISTAKES
================================================================================

  ❌ WRONG                              ✓ CORRECT
  ───────────────────────────────────────────────────────────────────────────
  I like the dogs. (general)            I like dogs.
  The life is beautiful. (general)      Life is beautiful.
  She speaks the Spanish.               She speaks Spanish.
  I go to work by the bus.              I go to work by bus.
  I go to the school every day.         I go to school every day.
  He plays the football.                He plays football.
  I live in the Spain.                  I live in Spain.
  She's doctor.                         She's a doctor.
  What beautiful day!                   What a beautiful day!
  The my book is here.                  My book is here.
  I had the breakfast.                  I had breakfast.
  The President Biden...                President Biden...
  I saw a moon.                         I saw the moon.

================================================================================
                    SPANISH vs ENGLISH
================================================================================

Spanish uses article, English doesn't:
┌───────────────────────────────────────────────────────────────────────────────┐
│ Me gusta el café.          →  I like coffee. (NOT the coffee)                 │
│ La vida es bella.          →  Life is beautiful. (NOT the life)               │
│ Los perros son leales.     →  Dogs are loyal. (NOT the dogs)                  │
│ Habla español.             →  He speaks Spanish. (NOT the Spanish)            │
│ Juega al tenis.            →  He plays tennis. (NOT the tennis)               │
│ El señor García            →  Mr. García (NOT the Mr. García)                 │
│ La reina Isabel            →  Queen Elizabeth (NOT the Queen Elizabeth)       │
│ Voy al colegio.            →  I go to school. (NOT the school)                │
│ En casa                    →  At home (NOT at the home)                       │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    DECISION FLOWCHART
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. Is it UNCOUNTABLE or PLURAL and GENERAL?                                 │
│    YES → NO ARTICLE: Coffee is good. Dogs are loyal.                        │
│                                                                             │
│ 2. Is it SINGULAR COUNTABLE?                                                │
│    • First mention / one of many → A/AN: I saw a dog.                       │
│    • Specific / already known → THE: The dog was brown.                     │
│                                                                             │
│ 3. Is it UNIQUE (sun, moon) or SUPERLATIVE?                                 │
│    YES → THE: The sun, the best                                             │
│                                                                             │
│ 4. Is it a LANGUAGE, SPORT, MEAL, or TRANSPORT?                             │
│    YES → NO ARTICLE: speak French, play tennis, have lunch, by bus          │
│                                                                             │
│ 5. Is it a PLACE (function) like school, work, home, bed?                   │
│    YES → NO ARTICLE: go to school, at home, in bed                          │
│                                                                             │
│ 6. Is it a COUNTRY, CITY, or CONTINENT?                                     │
│    Most → NO ARTICLE: Spain, Paris, Europe                                  │
│    (Except: the USA, the UK, the Netherlands)                               │
└─────────────────────────────────────────────────────────────────────────────┘
