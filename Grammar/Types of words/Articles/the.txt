================================================================================
                    THE DEFINITE ARTICLE: THE
                    (El Artículo Definido: THE)
================================================================================

THE es el artículo definido. Se usa cuando hablamos de algo ESPECÍFICO
o que ya conocemos. Equivale a "el, la, los, las" en español.

================================================================================
                    PRONUNCIATION
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ /ðə/  → before CONSONANT sounds:  the book, the car, the university        │
│ /ðiː/ → before VOWEL sounds:      the apple, the hour, the umbrella        │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    WHEN TO USE THE
================================================================================

1. SPECIFIC / ALREADY MENTIONED:
┌───────────────────────────────────────────────────────────────────────────────┐
│ I saw a dog. The dog was brown.                                               │
│ Vi un perro. El perro era marrón.                                             │
│ (first mention = a, second mention = the)                                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ Where's the book I lent you?                                                  │
│ ¿Dónde está el libro que te presté?                                           │
│ (specific book - both know which one)                                         │
├───────────────────────────────────────────────────────────────────────────────┤
│ Can you close the door?                                                       │
│ ¿Puedes cerrar la puerta?                                                     │
│ (the door of this room - obvious which one)                                   │
└───────────────────────────────────────────────────────────────────────────────┘

2. UNIQUE THINGS (only one exists):
┌───────────────────────────────────────────────────────────────────────────────┐
│ the sun       the moon       the Earth       the sky                         │
│ the world     the universe   the equator     the internet                    │
│ the environment              the government  the economy                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ The sun rises in the east.                                                    │
│ El sol sale por el este.                                                      │
├───────────────────────────────────────────────────────────────────────────────┤
│ The moon is beautiful tonight.                                                │
│ La luna está preciosa esta noche.                                             │
└───────────────────────────────────────────────────────────────────────────────┘

3. SUPERLATIVES:
┌───────────────────────────────────────────────────────────────────────────────┐
│ She's the best student in the class.                                          │
│ Es la mejor estudiante de la clase.                                           │
├───────────────────────────────────────────────────────────────────────────────┤
│ It's the most expensive hotel in town.                                        │
│ Es el hotel más caro de la ciudad.                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ He's the tallest person I know.                                               │
│ Es la persona más alta que conozco.                                           │
└───────────────────────────────────────────────────────────────────────────────┘

4. ORDINAL NUMBERS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the first     the second    the third     the last                            │
│ the next      the only      the same                                          │
├───────────────────────────────────────────────────────────────────────────────┤
│ He was the first person to arrive.                                            │
│ Fue la primera persona en llegar.                                             │
├───────────────────────────────────────────────────────────────────────────────┤
│ This is the only solution.                                                    │
│ Esta es la única solución.                                                    │
└───────────────────────────────────────────────────────────────────────────────┘

5. GEOGRAPHICAL NAMES:

WITH THE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ RIVERS:       the Thames, the Amazon, the Nile                                │
│ SEAS/OCEANS:  the Mediterranean, the Atlantic, the Pacific                    │
│ MOUNTAIN RANGES: the Alps, the Himalayas, the Andes                           │
│ DESERTS:      the Sahara, the Gobi                                            │
│ ISLAND GROUPS: the Canary Islands, the Philippines                            │
│ COUNTRIES (plural/with Republic/Kingdom/States):                              │
│               the USA, the UK, the Netherlands, the Czech Republic            │
│ REGIONS:      the Middle East, the North, the South                           │
└───────────────────────────────────────────────────────────────────────────────┘

WITHOUT THE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ CONTINENTS:   Europe, Asia, Africa, Australia (NOT the Europe)                │
│ MOST COUNTRIES: Spain, France, Japan, Brazil (NOT the Spain)                  │
│ CITIES:       London, Paris, Tokyo (NOT the London)                           │
│ SINGLE MOUNTAINS: Mount Everest, Mount Fuji (NOT the Mount Everest)           │
│ SINGLE ISLANDS: Sicily, Mallorca (NOT the Sicily)                             │
│ LAKES:        Lake Victoria, Lake Titicaca (NOT the Lake Victoria)            │
└───────────────────────────────────────────────────────────────────────────────┘

6. BUILDINGS AND INSTITUTIONS:

WITH THE:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the Eiffel Tower    the White House    the Empire State Building              │
│ the Louvre          the British Museum the Statue of Liberty                  │
│ the Ritz Hotel      the Times          the BBC                                │
└───────────────────────────────────────────────────────────────────────────────┘

7. MUSICAL INSTRUMENTS (playing):
┌───────────────────────────────────────────────────────────────────────────────┐
│ She plays the piano.                                                          │
│ Ella toca el piano.                                                           │
├───────────────────────────────────────────────────────────────────────────────┤
│ He's learning the guitar.                                                     │
│ Está aprendiendo la guitarra.                                                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ I can play the violin.                                                        │
│ Sé tocar el violín.                                                           │
└───────────────────────────────────────────────────────────────────────────────┘

8. ADJECTIVES AS NOUNS (groups of people):
┌───────────────────────────────────────────────────────────────────────────────┐
│ the rich (= rich people)        the poor (= poor people)                      │
│ los ricos                       los pobres                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ the young        the old        the elderly       the homeless                │
│ the sick         the injured    the unemployed    the disabled                │
├───────────────────────────────────────────────────────────────────────────────┤
│ The rich are getting richer.                                                  │
│ Los ricos se están haciendo más ricos.                                        │
└───────────────────────────────────────────────────────────────────────────────┘

9. NATIONALITIES (as a group):
┌───────────────────────────────────────────────────────────────────────────────┐
│ the Spanish (= Spanish people)                                                │
│ los españoles                                                                 │
├───────────────────────────────────────────────────────────────────────────────┤
│ the French      the British     the Dutch       the Japanese                  │
│ the Chinese     the Swiss       the Irish       the Welsh                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ The Japanese are known for their technology.                                  │
│ Los japoneses son conocidos por su tecnología.                                │
└───────────────────────────────────────────────────────────────────────────────┘

10. DECADES AND HISTORICAL PERIODS:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the 1960s       the 1990s       the 21st century                              │
│ the Middle Ages the Renaissance the Industrial Revolution                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ Music was different in the 1980s.                                             │
│ La música era diferente en los años 80.                                       │
└───────────────────────────────────────────────────────────────────────────────┘

11. PARTS OF THE DAY (with "in"):
┌───────────────────────────────────────────────────────────────────────────────┐
│ in the morning     in the afternoon    in the evening                         │
│ por la mañana      por la tarde        por la noche/tarde                     │
├───────────────────────────────────────────────────────────────────────────────┤
│ BUT: at night, at noon, at midnight (no THE)                                  │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    SPECIAL CASES
================================================================================

THE + NOUN + OF:
┌───────────────────────────────────────────────────────────────────────────────┐
│ the beginning of the movie                                                    │
│ el principio de la película                                                   │
├───────────────────────────────────────────────────────────────────────────────┤
│ the end of the book                                                           │
│ el final del libro                                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│ the capital of France                                                         │
│ la capital de Francia                                                         │
└───────────────────────────────────────────────────────────────────────────────┘

THE + SAME:
┌───────────────────────────────────────────────────────────────────────────────┐
│ We have the same teacher.                                                     │
│ Tenemos el mismo profesor.                                                    │
├───────────────────────────────────────────────────────────────────────────────┤
│ It's the same problem.                                                        │
│ Es el mismo problema.                                                         │
└───────────────────────────────────────────────────────────────────────────────┘
