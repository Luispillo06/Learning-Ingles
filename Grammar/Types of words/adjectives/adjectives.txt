================================================================================
                            ADJECTIVES
                      (Adjetivos en Inglés)
================================================================================

Los adjetivos son palabras que DESCRIBEN o MODIFICAN sustantivos.
Nos dicen cómo es algo o alguien.

================================================================================
                    POSITION OF ADJECTIVES
================================================================================

1. BEFORE THE NOUN (antes del sustantivo) - más común:

┌───────────────────────────────────────────────────────────────────────────────┐
│ a beautiful house                 → una casa bonita                           │
│ an old car                        → un coche viejo                            │
│ the big dog                       → el perro grande                           │
│ my new phone                      → mi teléfono nuevo                         │
│ three small children              → tres niños pequeños                       │
└───────────────────────────────────────────────────────────────────────────────┘

2. AFTER THE VERB (después del verbo) - con verbos de estado:

┌───────────────────────────────────────────────────────────────────────────────┐
│ She is beautiful.                 → Ella es guapa.                            │
│ The car looks old.                → El coche parece viejo.                    │
│ I feel tired.                     → Me siento cansado.                        │
│ The food tastes delicious.        → La comida sabe deliciosa.                 │
│ He seems happy.                   → Él parece feliz.                          │
└───────────────────────────────────────────────────────────────────────────────┘

Verbos comunes + adjetivo:
be, seem, appear, look, feel, sound, taste, smell, become, get

================================================================================
                    ORDER OF ADJECTIVES
================================================================================

Cuando usamos varios adjetivos, siguen este ORDEN:

┌─────────────────────────────────────────────────────────────────────────────┐
│ OPINION → SIZE → AGE → SHAPE → COLOUR → ORIGIN → MATERIAL → PURPOSE        │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Opinion:    beautiful, ugly, nice, lovely, horrible, amazing             │
│ 2. Size:       big, small, tall, short, long, huge, tiny                    │
│ 3. Age:        old, new, young, ancient, modern                             │
│ 4. Shape:      round, square, flat, rectangular, oval                       │
│ 5. Colour:     red, blue, green, black, white, dark, light                  │
│ 6. Origin:     Spanish, British, American, Chinese, Italian                 │
│ 7. Material:   wooden, metal, plastic, glass, cotton, silk                  │
│ 8. Purpose:    sleeping (bag), running (shoes), cooking (oil)               │
└─────────────────────────────────────────────────────────────────────────────┘

EXAMPLES:

→ a beautiful big old house (opinion + size + age)
→ a lovely little Italian restaurant (opinion + size + origin)
→ an ugly old brown leather bag (opinion + age + colour + material)
→ a small round wooden table (size + shape + material)
→ those expensive new running shoes (opinion + age + purpose)

⚠️ Normalmente no usamos más de 2-3 adjetivos juntos.

================================================================================
                    2. POSSESSIVE ADJECTIVES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ my          → mi/mis                                                        │
│ your        → tu/tus (o su/sus de usted)                                    │
│ his         → su/sus (de él)                                                │
│ her         → su/sus (de ella)                                              │
│ its         → su/sus (de ello - cosas/animales)                             │
│ our         → nuestro/a/os/as                                               │
│ their       → su/sus (de ellos/ellas)                                       │
└─────────────────────────────────────────────────────────────────────────────┘

→ This is my book.
→ Her name is Sarah.
→ Their house is big.

================================================================================
                    3. DEMONSTRATIVE ADJECTIVES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ this        → este/esta (singular, cerca)                                   │
│ that        → ese/esa/aquel/aquella (singular, lejos)                       │
│ these       → estos/estas (plural, cerca)                                   │
│ those       → esos/esas/aquellos/as (plural, lejos)                         │
└─────────────────────────────────────────────────────────────────────────────┘

→ This book is interesting.
→ That car is expensive.
→ These shoes are new.
→ Those houses are old.

================================================================================
                    4. QUANTITATIVE ADJECTIVES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ some        → algún, algo de      │ any         → algún, ningún             │
│ many        → muchos              │ much        → mucho                     │
│ few         → pocos               │ little      → poco                      │
│ a few       → unos pocos          │ a little    → un poco                   │
│ several     → varios              │ enough      → suficiente                │
│ all         → todo/todos          │ no          → ningún                    │
│ most        → la mayoría          │ each        → cada                      │
│ every       → cada, todo          │ both        → ambos                     │
└─────────────────────────────────────────────────────────────────────────────┘

→ I have some money.
→ There are many people here.
→ She has few friends.
→ We have enough time.

================================================================================
                    5. INTERROGATIVE ADJECTIVES
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ which       → cuál, qué                                                     │
│ what        → qué                                                           │
│ whose       → de quién                                                      │
└─────────────────────────────────────────────────────────────────────────────┘

→ Which book do you want?
→ What colour is your car?
→ Whose bag is this?

================================================================================
                    6. NUMERAL ADJECTIVES
================================================================================

CARDINAL (Cardinales):
→ one, two, three, four, five...
→ I have three cats.

ORDINAL (Ordinales):
→ first, second, third, fourth, fifth...
→ This is my first time here.

================================================================================
                    -ED vs -ING ADJECTIVES
================================================================================

Esta es una distinción MUY importante:

┌───────────────────────────────────────────────────────────────────────────────┐
│ -ED ADJECTIVES: describe how someone FEELS (cómo se siente alguien)          │
│ -ING ADJECTIVES: describe what CAUSES the feeling (qué causa el sentimiento) │
└───────────────────────────────────────────────────────────────────────────────┘

┌──────────────────┬──────────────────┬─────────────────────────────────────────┐
│ -ED (FEELINGS)   │ -ING (CAUSES)    │ MEANING                                 │
├──────────────────┼──────────────────┼─────────────────────────────────────────┤
│ bored            │ boring           │ aburrido / aburrido (que aburre)        │
│ interested       │ interesting      │ interesado / interesante                │
│ excited          │ exciting         │ emocionado / emocionante                │
│ tired            │ tiring           │ cansado / cansador                      │
│ amazed           │ amazing          │ asombrado / asombroso                   │
│ surprised        │ surprising       │ sorprendido / sorprendente              │
│ confused         │ confusing        │ confundido / confuso                    │
│ frightened       │ frightening      │ asustado / aterrador                    │
│ relaxed          │ relaxing         │ relajado / relajante                    │
│ annoyed          │ annoying         │ molesto / molesto (que molesta)         │
│ disappointed     │ disappointing    │ decepcionado / decepcionante            │
│ frustrated       │ frustrating      │ frustrado / frustrante                  │
│ embarrassed      │ embarrassing     │ avergonzado / vergonzoso                │
│ satisfied        │ satisfying       │ satisfecho / satisfactorio              │
│ shocked          │ shocking         │ conmocionado / impactante               │
│ exhausted        │ exhausting       │ agotado / agotador                      │
│ fascinated       │ fascinating      │ fascinado / fascinante                  │
│ depressed        │ depressing       │ deprimido / deprimente                  │
│ worried          │ worrying         │ preocupado / preocupante                │
│ entertained      │ entertaining     │ entretenido / entretenido               │
└──────────────────┴──────────────────┴─────────────────────────────────────────┘

EXAMPLES:

┌───────────────────────────────────────────────────────────────────────────────┐
│ The movie was boring.         → La película era aburrida.                     │
│ I was bored during the movie. → Yo estaba aburrido durante la película.       │
├───────────────────────────────────────────────────────────────────────────────┤
│ The news is surprising.       → La noticia es sorprendente.                   │
│ I'm surprised by the news.    → Estoy sorprendido por la noticia.             │
├───────────────────────────────────────────────────────────────────────────────┤
│ This book is interesting.     → Este libro es interesante.                    │
│ I'm interested in this book.  → Estoy interesado en este libro.               │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    ADJECTIVES vs ADVERBS
================================================================================

┌───────────────────────────────────────────────────────────────────────────────┐
│ ADJECTIVE: describe NOUNS (sustantivos)                                       │
│ ADVERB: describe VERBS, adjectives, or other adverbs                          │
└───────────────────────────────────────────────────────────────────────────────┘

ADJECTIVE + NOUN:
→ She is a careful driver. (adjective + noun)

VERB + ADVERB:
→ She drives carefully. (verb + adverb)

EXAMPLES:

┌───────────────────────────────────────────────────────────────────────────────┐
│ He is a slow walker. (adj)    → He walks slowly. (adv)                        │
│ She is a fast runner. (adj)   → She runs fast. (adv)                          │
│ It's a beautiful song. (adj)  → She sings beautifully. (adv)                  │
│ He's a good teacher. (adj)    → He teaches well. (adv)                        │
└───────────────────────────────────────────────────────────────────────────────┘

================================================================================
                    COMMON MISTAKES
================================================================================

  ❌ WRONG                              ✓ CORRECT
  ───────────────────────────────────────────────────────────────────────────
  I'm boring.                           I'm bored. (feeling)
  The movie was bored.                  The movie was boring. (cause)
  A car red.                            A red car. (adj before noun)
  She drives careful.                   She drives carefully. (adverb)
  A beautiful big old Italian car.      ✓ (correct order)
  He is very handsome man.              He is a very handsome man.
  I have news interesting.              I have interesting news.
